# Campus ATD Setup

## Getting Started

This repository contains the automation needed to configure a Campus ATD roadshow event. This include the following

- Configuration of all EOS devices using AVD
- Configuration of CV-CUE to AGNI radsec certificates using `cue_agni_radsec` role
- Configuration of EOS to AGNI radsec certificates using `eos_agni_radsec` role
- Configuration of AGNI base objects and policies
